Computer Vision Cheat Sheet

Quick reference for common image processing operations (OpenCV)

Image Preprocessing

Load Image

Read image into memory.

cv2.imread("image.png")

Grayscale

Convert RGB image into single intensity channel.

cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

Resize

Standardize image size for processing.

cv2.resize(img, (width,height))

ROI

Crop only region of interest.

roi = img[y1:y2, x1:x2]

Noise Removal

Gaussian Blur

Smooth image and reduce high-frequency noise.

cv2.GaussianBlur(img,(5,5),0)

Median Blur

Removes salt-and-pepper noise.

cv2.medianBlur(img,5)

Segmentation

Threshold

Convert grayscale image into binary image.

cv2.threshold(gray,120,255,cv2.THRESH_BINARY)

Adaptive Threshold

Threshold computed locally for uneven lighting.

cv2.adaptiveThreshold()

Edge Detection

Detect boundaries of objects.

cv2.Canny(gray,50,150)

Shape Detection

Contours

Extract object boundaries.

cv2.findContours()

Bounding Box

Draw rectangle around detected object.

cv2.boundingRect(cnt)

Connected Components

Label objects in binary image.

cv2.connectedComponents()

Morphological Operations

Erosion

Shrinks white objects. Removes noise.

cv2.erode()

Dilation

Expands white regions. Fills gaps.

cv2.dilate()

Opening

Erosion → Dilation (remove noise).

cv2.morphologyEx(img, cv2.MORPH_OPEN)

Closing

Dilation → Erosion (fill holes).

cv2.morphologyEx(img, cv2.MORPH_CLOSE)
